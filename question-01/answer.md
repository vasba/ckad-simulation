# Answer - Question 1 | Namespaces

```bash
kubectl get ns > namespaces
```

```
NAME               STATUS   AGE
default            Active   136m
earth              Active   105m
jupiter            Active   105m
kube-node-lease    Active   136m
kube-public        Active   136m
kube-system        Active   136m
mars               Active   105m
shell-intern       Active   105m
```
